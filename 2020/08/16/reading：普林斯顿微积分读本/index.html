读书笔记：普林斯顿微积分读本

微积分是大一第一学期的课程，本来学的就比较浅，而且在之后几年就没有任何使用的需求，所以水平越来越低。所以看普林斯顿微积分读本。

本书的目的是为了教大家微积分的主要概念以及求解题目的技巧，也可以用于一周一学期的备考。

主要内容

根据作者本书涉及的是单变量微积分的一些只是，主要是帮助大家通过考试。书中涉及的内容与高数比起来涉及的少了一点。

而书里涉及的内容按照先后顺序为：极限 -> 微分、导数 -> 积分 -> 反常积分 -> 级数、泰勒级数 -> 微分方程 ->其他话题。书里把泰勒级数和

特点

本书相当口语化，仿佛很明确地跟你说：我想让你搞懂然后让你考试通过。但是整本书看下来，其实只是看起来口语化而已，其实大部分的内容仍然是严谨的数学论述，该有的概念还有，例题仍然一大堆，它只是在书面上伪装了一下。本书真正让人感觉到亲切的是它引入概念的方式比较循序渐进，以及在把知识点搅碎喂给你的便利上。相对我看过的教材而言。

不是说其他教材难以理解，只是这本书太保姆了，体现非常鲜明的还有书中经常出现的解题流程图，第一第二第三第四按步骤写好，包你会。至少对新手而言这种方式非常优良，不仅包教包会，也是一个值得借鉴的学习方法。

体会

这本书给人的感觉是它不是给了你一本九阴真经，要你打通任督二脉，做出源源不断的难题。这本书确实更像救生圈，把你救上岸为止，接下来怎么跑就看你了。从小到大数学课程给我的感觉就是勇攀高峰，做出最难的题。所以数学书基本都变成了手册一样，给了你这些工具看你怎么做出难题来。但是微积分本身的熟悉光通过看 原理 + 例题 式的教材是有些痛苦的，这本微积分的基础教材很好地解决了这个问题，至少是一步步教下去的。

不过也是因为这个特点，我感觉这本书也不会适合已经熟悉微积分的人用来提高水平。毕竟如果已经上岸了就不需要lifesaver了。
